# 这是一个个人性质的组织，用于存放一些日用用于windows平台的源自vibe coding的小软件

[English](./README_EN.md) | 简体中文

具体列表如下:

[jks-viewer](https://github.com/yin2hao-windowsTools/jks-viewer)：一个使用Java开发的简单查看安卓.jks密钥库文件信息的软件

[portChecker](https://github.com/yin2hao-windowsTools/portChecker)：一个使用C#开发的快速查看windows端口占用，快速定位占用者并便捷释放的软件

[diskCompare](https://github.com/yin2hao-windowsTools/diskCompare)：一个使用C#开发的，保存磁盘占用快照，并将当前情况或快照与曾经的快照对比，快速找出存储空间变化罪魁祸首的软件

[blockScreen](https://github.com/yin2hao-windowsTools/blockScreen)：一个使用C# 开发的，将电脑屏幕置为黑色的软件
