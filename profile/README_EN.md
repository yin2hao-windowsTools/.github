# This is a personal organization used to store some small everyday Windows utilities created through vibe coding.

English | [简体中文](./README.md)

The repositories list is as follows:

[jks-viewer](https://github.com/yin2hao-windowsTools/jks-viewer): a simple Java-based tool for viewing information about Android `.jks` keystore files.

[portChecker](https://github.com/yin2hao-windowsTools/portChecker): a C# tool for quickly checking Windows port usage, identifying the occupying process, and conveniently releasing the port.

[diskCompare](https://github.com/yin2hao-windowsTools/diskCompare): a C# tool that saves disk usage snapshots and compares the current state or previous snapshots with earlier ones, helping quickly identify what caused changes in storage usage.

[blockScreen](https://github.com/yin2hao-windowsTools/blockScreen): a C# tool that turns the computer screen black.
